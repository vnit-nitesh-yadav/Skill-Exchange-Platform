## 🎯 Getting Started

### What is Skill Exchange Platform?

**Skill Exchange Platform** is a peer-to-peer learning community where CSE students can:
- 🎓 **Teach** skills they're good at
- 📚 **Learn** skills from experienced mentors
- 💬 **Connect** one-on-one with mentors
- ⭐ **Rate & Review** mentors and experiences
- 🚀 **Grow** together as a community

It's **100% FREE** - no money involved, just knowledge exchange!

## 👤 User Profile Setup

### Your Profile is Your Identity

Your profile is how others learn about you as a mentor or student. Make it detailed!

### Step-by-Step Profile Setup

#### **1. Basic Information**
- Navigate to **Profile** → **Edit Profile**
- Fill in these fields:
  - **Username**: Your display name (cannot change after signup)
  - **Email**: Your contact email
  - **Bio**: 50-200 characters about yourself
  - **Profile Picture**: Add a photo (JPG/PNG, max 5MB)

#### **2. Educational Background**
- **Department**: Select "CSE" if Computer Science
- **Graduation Year**: When you're graduating (e.g., 2025, 2026)
- **Experience Level**: 
  - 🟢 **Beginner**: < 1 year experience
  - 🟡 **Intermediate**: 1-2 years experience
  - 🟠 **Advanced**: 2-3 years experience
  - 🔴 **Expert**: 3+ years or certified

#### **3. Skills You Can Teach**
These are skills you're GOOD AT and can teach others:

**Example:**
```
Skill: React.js
Proficiency: Intermediate
Years of Experience: 1.5 years
Category: Frontend
```

**To Add Skills:**
1. Click "Add Skill" button
2. Enter skill name
3. Select proficiency level
4. Enter years of experience
5. Select category (Frontend, Backend, ML/AI, etc.)
6. Click "Save"

**Popular Skills Students Teach:**
- Frontend: HTML/CSS, JavaScript, React, Vue
- Backend: Node.js, Python, Django, Express
- Database: MongoDB, MySQL, PostgreSQL
- DevOps: Docker, Git, Linux, AWS
- DSA: Data Structures, Algorithms, C++
- ML/AI: Python, TensorFlow, Machine Learning

#### **4. Skills You Want to Learn**
These are skills you DON'T know yet but want to learn:

**Example:**
```
Skill: Machine Learning
Target Proficiency: Intermediate
Priority: High
```

**To Add Learning Goals:**
1. Click "Add Learning Goal" button
2. Enter skill name
3. Select target proficiency
4. Set priority (Low/Medium/High)
5. Click "Save"

#### **5. Learning Preferences**
- **Timezone**: Your timezone (for scheduling with mentors)
- **Preferred Learning Format:**
  - 📹 Video: Learn via video calls/tutorials
  - 📝 Text: Written guides and documents
  - 💬 1-on-1 Chat: Direct messaging
  - 🎯 Projects: Hands-on project-based learning
  - 👥 Pair Programming: Code together in real-time

#### **6. Social & Professional Links** (Optional)
- **GitHub Profile**: Link to your GitHub
- **LinkedIn Profile**: Link to your LinkedIn
- **Certifications**: Add any certificates/badges

## 🔍 Finding Skills & Mentors

### Two Ways to Find Mentors

#### **Method 1: Search by Skill**

1. Click **"Search Skills"** in the navigation
2. Enter the skill name in the search box:
   - "React"
   - "Python"
   - "Machine Learning"
   - etc.
3. Press Enter or click Search
4. **Results show:**
   - Mentor's name and profile picture
   - Average rating ⭐
   - Number of reviews
   - Skills they teach
5. Click on a mentor to view their full profile

#### **Method 2: Browse Mentors by Department/Level**

1. Click **"Recommended Mentors"**
2. **Filter options:**
   - Department (CSE, ECE, Mechanical)
   - Experience Level (Beginner, Intermediate, Advanced)
   - Skills offered
3. Mentors are sorted by rating and reviews
4. Click on a mentor to view details

### What to Look for in a Mentor

✅ **Good Mentor Indicators:**
- High rating (4.5-5 stars)
- Multiple reviews (5+ reviews)
- Clear skill description
- Responsive profile (recent activity)
- Teaching format matches your preference

## 🤝 Connecting with Others

### How to Send a Connection Request

1. **Find a mentor** (from search or recommendations)
2. Click **"Connect"** button
3. **Write a personalized message** (optional but recommended):
   - Why you want to learn from them
   - What you hope to achieve
   - When you're available
4. Click **"Send Request"**

**Example message:**
```
"Hi! I'm interested in learning React.js. 
I have basic JavaScript knowledge and am working on 
a web project. Would you be available for 
some 1-on-1 chat sessions this month?"
```

### Receiving Connection Requests

1. Go to **"Pending Requests"**
2. See all requests received
3. **Options:**
   - ✅ Accept: Start chatting immediately
   - ❌ Reject: Politely decline
   - 👀 View Profile: Learn more before deciding

### Your Connections

1. Go to **"Connected Users"**
2. See all active connections (both as mentor and student)
3. Click on a connection to start chatting

## 💬 Real-Time Chat

### Starting a Chat

1. Go to **"Connected Users"**
2. Click on the person you want to chat with
3. The chat window opens immediately

### Using Chat Features

**Sending Messages:**
- Type your message
- Click "Send" or press Enter
- ✓ Message sent instantly!

**Chat Features:**
- ⏰ **Timestamps**: See when messages were sent
- 📍 **Read Status**: See if message is delivered/read
- 🔄 **Real-Time**: No need to refresh, messages appear instantly
- 📱 **Mobile-Friendly**: Chat works on all devices

### Chat Best Practices

✅ **DO:**
- Be respectful and professional
- Ask clear questions
- Share resources and links
- Schedule regular learning sessions
- Provide feedback and progress updates

❌ **DON'T:**
- Share personal information (phone, address)
- Ask for money or payment
- Spam or send irrelevant messages
- Harass or use abusive language

### Ending a Connection

1. Go to **"Connected Users"**
2. Click on the connection
3. Click **"End Connection"** (if needed)
4. Both users are notified

## ⭐ Reviews & Ratings

### How Reviews Help

🎯 **Purpose:** Build trust in the community

- Help new students find good mentors
- Motivate mentors to provide quality teaching
- Provide feedback for improvement

### How to Leave a Review

1. After learning from a mentor, go to their **Profile**
2. Scroll to **"Reviews & Ratings"** section
3. Click **"Write a Review"**
4. Fill in the form:
   - **Rating**: 1-5 stars ⭐
   - **Review Type**: Learning from them / Teaching with them
   - **Comment**: Your honest feedback (50-500 characters)
   - **Detailed Ratings** (optional):
     - Communication: How well they explained
     - Teaching Quality: How much you learned
     - Reliability: Did they show up on time?
     - Responsiveness: How quickly they replied?

5. Click **"Submit Review"**

### Example Review

```
Rating: ⭐⭐⭐⭐⭐ (5 stars)

Comment: "Amazing mentor! Nitesh explained React concepts 
very clearly with real examples. He's patient, responsive, 
and helped me build my first real project. Highly recommend!"

Communication: 5/5
Teaching Quality: 5/5
Reliability: 4/5
Responsiveness: 5/5
```

### Reading Reviews of Mentors

1. Visit a mentor's **Profile**
2. Scroll to **"Reviews & Ratings"**
3. Read what other students said
4. Look for patterns:
   - Consistently high ratings = reliable mentor
   - Recent reviews are important
   - Read both praise and criticism

## 💡 Tips & Best Practices

### For Students (Learning)

✅ **DO:**
1. **Complete your profile** - Mentors are more likely to connect with complete profiles
2. **Be specific** - "I want to learn React for building web apps" is better than just "React"
3. **Set clear goals** - What do you want to build? What's your deadline?
4. **Show initiative** - Share your projects and progress with mentors
5. **Respect time** - Schedule sessions in advance
6. **Ask good questions** - Specific questions get better answers
7. **Practice between sessions** - Don't just rely on mentors
8. **Provide reviews** - Help others find good mentors

❌ **DON'T:**
- Expect immediate replies 24/7
- Ask for shortcuts or cheating help
- Ghost after connecting (reply to messages!)
- Ignore feedback from mentors

### For Mentors (Teaching)

✅ **DO:**
1. **Keep profile updated** - Update skills and availability
2. **Be responsive** - Reply within 24 hours if possible
3. **Be clear** - Explain concepts, don't just give answers
4. **Share resources** - Send useful links and tutorials
5. **Be patient** - Everyone learns at different speeds
6. **Encourage growth** - Celebrate progress!
7. **Give timely feedback** - Help students improve
8. **Set boundaries** - It's okay to say "no" if busy

❌ **DON'T:**
- Overcommit yourself
- Be condescending
- Only give dry theory (use examples!)
- Demand payment for teaching

### Getting Most Out of Connections

**Session Structure (Recommended):**

1. **First Session (15-20 min):**
   - Introduce yourself
   - Discuss learning goals
   - Agree on schedule and format

2. **Regular Sessions (30-60 min):**
   - Review previous concepts
   - Teach new material
   - Work on projects/problems
   - Q&A

3. **After Session:**
   - Mentor provides practice tasks
   - Student works on assignments
   - Schedule next session

### Setting Up Productive Sessions

**Before Session:**
- Student prepares questions
- Review previous notes
- Have camera/mic ready (if video)

**During Session:**
- Take notes
- Ask clarifying questions
- Share screen (if coding)
- Record (if mentor agrees)

**After Session:**
- Review notes
- Do practice problems
- Implement learnings
- Prepare for next session

## 🆘 Troubleshooting

### Common Issues

**Q: I can't log in**
- Check your email and password are correct
- Verify you created an account first
- Clear browser cache and try again
- Contact support if still failing

**Q: Can't find mentors for my skill**
- Try different skill names
- Search related skills
- Check if mentors are online
- Consider learning a prerequisite skill first

**Q: Mentor not responding**
- Give them 24-48 hours
- Send a follow-up message (polite)
- Try another mentor
- Mentors have their own schedule

**Q: How to report inappropriate behavior?**
- Take screenshot of messages
- Email: support@skillexchange.com
- We take safety seriously ⚠️
